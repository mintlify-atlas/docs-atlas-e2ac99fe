`TableFrame` is Tabsdata's primary data-manipulation object. It wraps a Polars `LazyFrame` and adds Tabsdata-specific system columns, provenance tracking, and data-quality hooks. Like a Polars lazy frame, operations on a `TableFrame` are not executed immediatelyŌĆöthey build a logical plan that the engine executes when the function returns. All methods return a new `TableFrame`, making the API fully composable.

<Note>
Import the core API with `import tabsdata as td`. `TableFrame`, `col`, `lit`, and `concat` are available directly from `td`. Factory functions (`from_polars`, `from_pandas`, `from_dict`, `empty`) and advanced utilities (`UDF`, `selectors`, `Schema`) live in `tabsdata.tableframe` (`import tabsdata.tableframe as tdf`).
</Note>

---

## Factory functions

Use these module-level helpers to construct a `TableFrame` from external data. Inside a `@td.publisher`, `@td.transformer`, or `@td.subscriber` function the framework provides `TableFrame` arguments automaticallyŌĆöthese factories are mainly useful for tests and stand-alone scripts.

They live in the `tabsdata.tableframe` sub-module (not the top-level `td` namespace):

```python
import tabsdata.tableframe as tdf
```

<ParamField path="tdf.from_polars(data)" type="function">
  Create a `TableFrame` from a `polars.LazyFrame`, `polars.DataFrame`, or `None`.
  Passing `None` returns an empty (zero-column, zero-row) `TableFrame`.

  ```python
  import polars as pl
  import tabsdata.tableframe as tdf

  lf = pl.LazyFrame({"x": [1, 2, 3]})
  tf = tdf.from_polars(lf)
  ```
</ParamField>

<ParamField path="tdf.from_pandas(data)" type="function">
  Create a `TableFrame` from a `pandas.DataFrame` or `None`.

  ```python
  import pandas as pd
  import tabsdata.tableframe as tdf

  df = pd.DataFrame({"name": ["Alice", "Bob"], "score": [95, 87]})
  tf = tdf.from_pandas(df)
  ```
</ParamField>

<ParamField path="tdf.from_dict(data)" type="function">
  Create a `TableFrame` from a plain Python dictionary mapping column names to
  lists of values, or `None`.

  ```python
  import tabsdata.tableframe as tdf

  tf = tdf.from_dict({"city": ["NY", "LA"], "pop": [8_000_000, 4_000_000]})
  ```
</ParamField>

<ParamField path="tdf.empty()" type="function">
  Create a zero-column, zero-row `TableFrame`. Accepts an optional `schema`
  argument (a `tdf.Schema`) to define column types upfront.
</ParamField>

## Conversion functions

These are available as instance methods on the `TableFrame` object:

```python
df  = tf.to_polars_df()   # -> polars.DataFrame
lf  = tf.to_polars_lf()   # -> polars.LazyFrame
pdf = tf.to_pandas()       # -> pandas.DataFrame
d   = tf.to_dict()         # -> dict[str, list[Any]]
```

The same functions are also available as module-level helpers in `tabsdata.tableframe`:

```python
import tabsdata.tableframe as tdf

df  = tdf.to_polars_df(tf)   # -> polars.DataFrame
lf  = tdf.to_polars_lf(tf)   # -> polars.LazyFrame
pdf = tdf.to_pandas(tf)       # -> pandas.DataFrame
d   = tdf.to_dict(tf)         # -> dict[str, list[Any]]
```

### Selection and projection

```python
# Select specific columns or computed expressions
result = tf.select(td.col("name"), td.col("score").mul(2).alias("double_score"))

# Add or replace columns without changing row count
result = tf.with_columns(
    td.col("score").cast(td.Float64).alias("score_f"),
    td.lit(0).alias("rank"),
)

# Drop named columns
result = tf.drop("temp_col", "scratch")

# Rename columns using a mapping
result = tf.rename({"old_name": "new_name"})
```

<ParamField path=".select(*exprs, **named_exprs)" type="method ŌåÆ TableFrame">
  Project the `TableFrame` to the given columns or expressions.

  ```python
  tf.select(td.col("a"), td.col("b").mul(2).alias("bx2"))
  ```
</ParamField>

<ParamField path=".with_columns(*exprs, **named_exprs)" type="method ŌåÆ TableFrame">
  Add new columns or replace existing ones. All original columns are retained.
</ParamField>

<ParamField path=".drop(*columns)" type="method ŌåÆ TableFrame">
  Remove one or more columns by name.
</ParamField>

<ParamField path=".rename(mapping)" type="method ŌåÆ TableFrame">
  Rename columns using a `dict[str, str]` mapping of `old_name ŌåÆ new_name`.
</ParamField>

### Filtering

<ParamField path=".filter(*predicates)" type="method ŌåÆ TableFrame">
  Keep only rows where all predicates evaluate to `True`.

  ```python
  tf.filter(td.col("score") >= 90, td.col("region").is_in(["EU", "US"]))
  ```
</ParamField>

<ParamField path=".unique(subset=None, keep='any', maintain_order=False)" type="method ŌåÆ TableFrame">
  Remove duplicate rows. Optionally specify a `subset` of columns to consider.
</ParamField>

<ParamField path=".drop_nulls(subset=None)" type="method ŌåÆ TableFrame">
  Remove rows that contain null values in any (or a specified subset of) column(s).
</ParamField>

<ParamField path=".drop_nans(subset=None)" type="method ŌåÆ TableFrame">
  Remove rows that contain `NaN` values.
</ParamField>

### Type casting

<ParamField path=".cast(dtypes, *, strict=True)" type="method ŌåÆ TableFrame">
  Cast one or more columns to new data types. Pass a mapping of
  `column_name ŌåÆ dtype` or a single dtype to cast all columns.

### Joins

<ParamField path=".join(other, on=None, how='inner', *, left_on=None, right_on=None, suffix='_right', join_nulls=False, coalesce=None)" type="method ŌåÆ TableFrame">
  Join this `TableFrame` with `other`.

  - **`on`** ŌĆö column name(s) present in both frames.
  - **`how`** ŌĆö join strategy: `"inner"`, `"left"`, `"right"`, `"full"`, `"cross"`, `"semi"`, or `"anti"`.
  - **`left_on` / `right_on`** ŌĆö use when key column names differ between frames.

  ```python
  result = customers.join(orders, on="customer_id", how="left")
  ```
</ParamField>

### Grouping and aggregation

<ParamField path=".group_by(*by)" type="method ŌåÆ TableFrameGroupBy">


## Expression API

Expressions are column references or computed values passed to `select`, `filter`, `with_columns`, and other transformation methods.

<ParamField path="td.col(name)" type="function ŌåÆ Expr">
  Reference a column by name. Supports chained expression methods such as
  `.alias()`, `.cast()`, `.sum()`, `.is_in()`, `.mul()`, and the full Polars
  expression API.

  ```python
  td.col("price").mul(td.col("quantity")).alias("revenue")
  ```
</ParamField>

<ParamField path="td.lit(value)" type="function ŌåÆ Expr">
  Create a literal value expression. Useful for adding constant columns.

  ```python
  tf.with_columns(td.lit("USD").alias("currency"))
  ```
</ParamField>

<ParamField path="td.concat(frames, *, how='vertical')" type="function ŌåÆ TableFrame">
  Concatenate a list of `TableFrame` objects. Default is row-wise (`"vertical"`).
  Other strategies supported by Polars (`"diagonal"`, `"horizontal"`) are also
  accepted.

  ```python
  combined = td.concat([tf_jan, tf_feb, tf_mar])
  ```
</ParamField>

## Data types

All Polars data types are re-exported from `tabsdata` for convenience:

| Type | Description |
|------|-------------|
| `td.Boolean` | True / False |
| `td.String` | UTF-8 string |
| `td.Int8`, `td.Int16`, `td.Int32`, `td.Int64`, `td.Int128` | Signed integers |
| `td.UInt8`, `td.UInt16`, `td.UInt32`, `td.UInt64` | Unsigned integers |
| `td.Float16`, `td.Float32`, `td.Float64` | Floating-point numbers |
| `td.Date` | Calendar date (no time) |
| `td.Datetime` | Date + time, with optional time zone |
| `td.Duration` | Time span / interval |
| `td.Time` | Time-of-day |
| `td.Decimal` | Arbitrary-precision decimal |
| `td.Categorical` | Low-cardinality string category |
| `td.Enum` | Fixed set of string values |
| `td.Null` | Null / missing type |

**Type groups** (sets of related types) are also available:

```python
td.FLOAT_DTYPES          # {Float16, Float32, Float64}
td.INTEGER_DTYPES        # all integer types
td.SIGNED_INTEGER_DTYPES # {Int8, Int16, Int32, Int64, Int128}
td.UNSIGNED_INTEGER_DTYPES # {UInt8, UInt16, UInt32, UInt64}
td.NUMERIC_DTYPES        # integers + floats
td.TEMPORAL_DTYPES       # {Date, Datetime, Duration, Time}
```

## Column selectors ŌĆö `tdf.selectors`

`tabsdata.tableframe.selectors` provides helper utilities for selecting columns by type or pattern without naming them explicitly.

```python
import tabsdata.tableframe as tdf

# Select all numeric columns
tf.select(tdf.selectors.numeric())

# Select all string columns
tf.select(tdf.selectors.string())

# Combine selectors
tf.select(tdf.selectors.numeric() | tdf.selectors.by_name("id"))
```

## User-defined functions ŌĆö `tdf.UDF`

The `UDF` base class (in `tabsdata.tableframe`) lets you apply custom Python logic element-by-element or batch-by-batch via the `.udf()` method on a `TableFrame`. Subclass `tdf.UDF` and implement `on_element` or `on_batch`.

The `.udf()` method takes two positional arguments: `on` (an expression or list of expressions selecting input columns) and `function` (the `UDF` instance).

```python
import tabsdata as td
import tabsdata.tableframe as tdf

class ScoreNormalizer(tdf.UDF):
    def __init__(self):
        super().__init__([("norm_score", td.Float64)])

    def on_element(self, score):
        return [min(max(score / 100.0, 0.0), 1.0)]

result = tf.udf(on=td.col("score"), function=ScoreNormalizer())
```

## Complete example

```python
import tabsdata as td

@td.transformer(
    input_tables=["orders", "products"],
    output_tables="enriched_orders",
)
def enrich_orders(
    orders: td.TableFrame,
    products: td.TableFrame,
) -> td.TableFrame:
    return (
        orders
        .join(products, on="product_id", how="left")
        .filter(td.col("status") == "shipped")
        .with_columns(
            td.col("price").mul(td.col("qty")).alias("revenue"),
        )
        .select("order_id", "customer_id", "product_name", "revenue")
        .sort("revenue", descending=True)
    )
```
